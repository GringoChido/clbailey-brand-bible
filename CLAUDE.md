# CLAUDE.md — C.L. Bailey & Co. Brand Operating Instructions

**This is a C.L. Bailey & Co. brand project. The Brand Bible in this folder is the single source of truth for every creative decision. Do not go outside it.**

## Before producing ANYTHING, do this first

1. Read `GUARDRAILS.md` — the enforceable MUST / NEVER rules and the standing guardrail prompt (§0).
2. For anything the guardrails don't fully cover, consult `spec/BRAND_BIBLE.md` (the full 11-module specification) and `spec/PHOTOGRAPHY.md` (the image standard).
3. Treat `portal/index.html` + `portal/style.css` as the canonical design-token reference (colors, type, spacing).

Apply this to all creative output: copy, social posts, ads, emails, layouts, image generation, photography direction, co-brand lockups, and web/portal work — English and Spanish.

## The rule

Every output must trace back to a guideline. If a creative choice can't be traced to the Brand Bible, it is off-brand by definition — ask for the spec or omit it. **Never invent facts, collections, finishes, dimensions, or guarantees.**

## Self-check before you hand anything back

Run the 60-Second Review Checklist (GUARDRAILS.md §8) against the output. The non-negotiables most often missed:

- **Material truth:** solid hardwood only — never MDF/particle board/veneer (except the one sanctioned "No MDF" contrast claim, used verbatim). State specs exactly: 1-inch diamond-honed slate, 8 hand-applied finishes.
- **Color:** Warm White `#FAF8F5` grounds, 60/30/10 proportion, Bronze ≤5%. Heritage Green `#006840` appears ONLY in the mark and the felt.
- **Type:** Le Jour Serif 400 headlines (never bolded, never <19px); Roboto body; Raleway uppercase labels/buttons (+2px tracking).
- **Mark:** never recolor, stretch, outline, or rebuild. Only the Primary badge exists in production — never improvise a Lockup or Crest.
- **Voice:** master carpenter explaining their work — premium not elitist, traditional not antiquated, durable not industrial. Oxford comma always; max one exclamation point (zero in spec/warranty).
- **CTA:** always "Find a Dealer," never "Shop Now." Label any price as MSRP; never strikethrough/below-MSRP in brand or co-op creative.
- **Spanish:** usted register always; locked Module 3-B translations only; accents on capitals; no machine translation ships unreviewed.
- **AI cliché ban-list** (GUARDRAILS §6): delve, testament, revolutionize, seamless, elevate, unlock, game-changer, tapestry, robust, boasts, etc. Write plainly.
- **AI imagery:** never passed off as a real product photo in dealer-facing or paid assets; workshop/makers are never AI-generated.
- **File naming:** `[Brand]-[Module]-[AssetType]-[Color]-[Version].[Ext]`.

If any output violates a MUST, fix it before presenting — no discretion.

## When asked, state which guideline you applied

When you make a notable brand call, name the rule behind it (e.g. "headline set in Le Jour Serif per §2 type hierarchy") so the decision is auditable.
