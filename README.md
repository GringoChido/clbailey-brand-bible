# C.L. Bailey & Co. — Brand Bible & Guidelines Engine

A complete corporate identity system for **C.L. Bailey & Co.** (handcrafted pool tables & game room furniture, est. 1999, Tomball TX) and its Authorized Dealer Network.

**Tagline:** Built for Homes That Endure. · **Palette:** Warm White `#FAF8F5` · Graphite `#3A3A3A` · Deep Charcoal `#1C1C1E` · Aged Bronze `#A89279` — Heritage Green `#006840` reserved for the mark and the felt. Mirrors the live clbailey.com design system.

## Using the portal

Open `portal/index.html` in any browser — no build step, no frameworks, no dependencies. Vanilla HTML/CSS/JS only. The palette grid and Dealer File Vault are interactive (click swatches to copy HEX). Designed to be hosted at `dealers.clbailey.com` as an authenticated asset page (`noindex` is set).

## Status

Standalone, actively-edited project. This is the single source of truth for the C.L. Bailey & Co. brand — independent of any other project or dashboard. `GUARDRAILS.md` is the enforceable layer (paste-at-session-start rules); `spec/BRAND_BIBLE.md` is the full reference; `portal/` is the live engine. As of v2.0 the design tokens mirror the live clbailey.com build exactly — the site and the bible are one system.
